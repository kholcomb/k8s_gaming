## Check Node Taints
```bash
kubectl describe node kind-control-plane | grep Taints
```
You'll see: `Taints: dedicated=gpu:NoSchedule`

## The Problem
- Node has taint: `dedicated=gpu:NoSchedule`
- Pod needs matching toleration
- Without toleration, pod can't schedule

## The Fix
Add toleration to pod spec matching the taint.

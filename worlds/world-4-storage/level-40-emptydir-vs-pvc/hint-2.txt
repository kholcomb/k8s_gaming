- Temporary cache
- Data sharing between containers in same pod

**PersistentVolumeClaim (Persistent):**
```yaml

- name: data
  persistentVolumeClaim:
    claimName: app-data  # âœ… For persistent data!
```

Characteristics:
- Backed by PersistentVolume
- **Survives pod deletion/restart**
- Data persists independently of pod lifecycle
- Can be reattached to new pods
- May have performance overhead (network storage)

Use cases:
- Databases
- User uploads
- Application state
- Logs that must be retained
- Any data that must survive pod restarts
